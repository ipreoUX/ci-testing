    modifiers

    @mixin modifiers($attribute:  , $prefix: -, $separator: -) { ... }
    View source

    Description

    Generate modifier classes based parent. Can get any value from the map, generate the classes with the style attribute.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $attribute

    css attribute target

    		Variable
    $prefix

    How you seperate the parent class and generated modifiers .c-text

    		Variable-
    $separator

    If you follow BEM and wish to use something like "--" to get .color-text--red

    		Variable-

    Example

    //Useage:
.parent {
  @include modifiers(shadows, 'box-shadow');
 }
      //Output:
.parent-xl {
  box-shadow: 0 12px 24px 0 rgba(0, 0, 0, 0.1); }

.parent-lg {
  box-shadow: 0 8px 16px 0 rgba(0, 0, 0, 0.1); }

.parent-md {
  box-shadow: 0 6px 12px 0 rgba(0, 0, 0, 0.1); }

.parent-sm {
  box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.1); }

.parent-xs {
  box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.1); }
